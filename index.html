Perangkat Jaringan

Pembahasan mendalam mengenai satu perangkat perantara dalam jaringan komputer.

Intermediary Device

Router

Fungsi Utama

Router adalah perangkat keras jaringan yang berfungsi untuk menghubungkan dua atau lebih jaringan yang berbeda (network ID berbeda). Router bekerja pada Layer 3 (Network Layer) dalam model OSI.

Fungsi utamanya adalah Routing, yaitu menentukan jalur terbaik (best path) untuk mengirimkan paket data dari satu jaringan ke jaringan lain agar sampai ke tujuan dengan efisien.

Jenis-Jenis Router

Wireless Router

Menyediakan akses konektivitas nirkabel (Wi-Fi) ke perangkat pengguna.

Core Router

Router berkapasitas tinggi yang menjadi tulang punggung (backbone) jaringan internet.

Edge Router

Berada di perbatasan jaringan, menghubungkan LAN lokal ke internet (ISP).
